    Sonar wraps the HTML5/Javascript NDT client originally developed by M-Lab into a more consistent and branded interface. This application uses the following libraries:

    • Twitter Bootstrap
    • jQuery
    • canv-guage
    • Raphael
    • morris.js Configuration and branding options can be found in the Sonar.app.config file. Module level parameters in this file should be left alone unless you really know what you are doing.

    An array of download test measurements. On each {@link #monitorTest} callback, the current reported value of the download speed is pushed to this array. Later, in the results, this array can be analyzed for average/max/min speed as desired. This differs from the {@link Sonar.client.NDTjs} reported speed, which appears to simply be the last reported speed and not necessarily the one we wish to report.
